# Agentic IT Support Desk

Agentic IT support system built for SJSU – Agentic AI course.

## What it does
- Takes user IT issues (VPN, WiFi, etc.)
- Routes through an agent workflow
- Pulls solutions from a Supabase knowledge base
- Applies escalation logic if needed

## Current Progress
- ✅ Intake agent working
- ✅ Knowledge agent connected to Supabase
- ✅ Environment variables secured with .env
- ✅ Basic routing + escalation flow
